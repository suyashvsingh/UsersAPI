Getting Started

Create a new user

  • POST request to https://svs-usersapi.herokuapp.com/users
  • Body should contain name, email and password keys
  • New user will be created successfully

Login

  • GET request to https://svs-usersapi.herokuapp.com/users
  • Body should contain email and password keys
  • User will receive his/her JSON web token

Updating information

  • PUT request to https://svs-usersapi.herokuapp.com/users
  • Body should contain updated info, with one or more fields, and Headers should have authorization field and value should be Bearer JWT, where JWT will be the JSON web token of the user
  • User will receive the updated information

Deleting

  • DELETE request to https://svs-usersapi.herokuapp.com/users
  • Headers should have authorization field and value should be Bearer JWT, where JWT will be the JSON web token of the user
  • The user will be deleted
